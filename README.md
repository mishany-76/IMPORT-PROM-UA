# IMPORT-PROM-UA 📦

Скрипт для автоматизации импорта товаров на платформу **Prom.ua** с использованием данных из **Google Sheets** и внешних XML/YML фидов.

## 🚀 Возможности
* Синхронизация остатков и цен из Google Таблиц.
* Обработка XML/YML фидов поставщиков.
* Автоматическое обновление данных по API Prom.ua.

## 🛠 Настройка перед использованием

Репозиторий настроен безопасно: все конфиденциальные данные вынесены в игнорируемые файлы. Для работы скрипта вам необходимо создать их локально:

### 1. Доступы к Prom.ua (`user.txt`)
Создайте в корне проекта файл `user.txt`. В нем должны храниться ваши авторизационные данные (например, логин, пароль или API-токен в зависимости от вашей реализации).
> **Важно:** Этот файл добавлен в `.gitignore` и не попадет в публичный доступ.

### 2. Google Sheets API (`key_sheet.json`)
Для работы с таблицами вам понадобится сервисный аккаунт Google:
1. Создайте проект в [Google Cloud Console](https://console.cloud.google.com/).
2. Включите **Google Sheets API**.
3. Создайте **Service Account**, скачайте ключ в формате JSON.
4. Переименуйте его в `key_sheet.json` и положите в корень проекта.

### 3. Настройка окружения
